# libp2p Documentation Portal

## Modular networking stack

libp2p is a modular system of protocols, specifications and libraries that enable the development of peer-to-peer network applications.
There are many distributed peer-to-peer network models with different challenges and tradeoffs that try to improve on the way we network.
Libp2p aims to be a modular, general-purpose toolkit for any peer-to-peer application.
